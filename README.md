# Blinq Generated Tests

This project contains Cucumber feature files and Playwright-backed step definitions generated by Blinq.

## Project Structure

```text
.
├── package.json
├── README.md
├── cucumber.mjs
├── environments.json
├── test-data.json
└── features/
    ├── *.feature
    └── step_definitions/
        ├── support/
        │   └── bvt.mjs
        └── *.mjs
```

- `features/` contains generated Cucumber feature files.
- `features/step_definitions/` contains generated JavaScript step definitions.
- `features/step_definitions/support/bvt.mjs` configures the Blinq Cucumber World and hooks.
- `environments.json` maps Blinq environment names to their base URLs.
- `test-data.json` stores generated global and environment-scoped seed test data.

## Install

```bash
npm install
npm run playwright:install
```

## Run Tests

Run the generated Cucumber tests:

```bash
npm test
```

Run against a specific Blinq environment:

```bash
BLINQ_ENVIRONMENT=staging npm test
```

Run an example Cucumber command against the generated features:

```bash
npm run test:example
```
